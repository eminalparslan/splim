Splim - Slice album mp3 file into individual songs

Usage:

  `splim [options] [-t=timestamps_file] [-i=input_file] [-o=output_directory]`
  
Options:

  `-t | --timestamps : timestamp file`
  
  `-i | --input      : album audio file`
  
  `-o | --output     : output directory`
  
  `-h | --help       : show help`
